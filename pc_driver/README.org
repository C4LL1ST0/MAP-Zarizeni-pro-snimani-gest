* pc_driver - ovládací program pro zařízení na snímání gest
Jednoduché terminálové prostředí umožnující připojení k zařízení na snímání gest, jeho využívání a sběr trénovacích dat pro nová gesta.

** RUN
 - požadavky: anaconda distribuce
  #+begin_src bash
   cd MAP-Zarizeni-pro-snimani-gest/pc_driver

   conda env create -f environment.yml
   conda activate pc_driver

   cd src
   python -m pc_driver.main
 #+end_src
